# MassiliaDrive - Website

## Project Description

This is a showcase website for a VTC (Vehicle with Driver) service in the Marseille region. The website aims to increase online visibility and reach top positions in Google search results for VTC transport-related keywords in Marseille.

## Features

### Main Sections

- **Hero Section**: Service presentation with call-to-action
- **About Section**: Company introduction and experience
- **Services Section**: Complete VTC services (Airport transfer, Business transport, Events, City rides, Tourist tours)
- **Vehicles Section**: Premium fleet showcase (Mercedes, BMW, Audi, Tesla)
- **Drivers Section**: Professional driver team presentation
- **Coverage Area**: Service zones in Marseille and PACA region
- **Reviews Section**: Customer testimonials
- **Reservation Section**: Simple booking form
- **Contact Section**: Contact information and booking

### Target Customers

- Airport travelers
- Professionals and businesses
- Event customers: weddings, parties, etc.

### Key Services

1. **Airport Transfer** - 24/7 shuttle service from/to Marseille Provence Airport
2. **Professional Transport** - Business appointments and corporate travel
3. **Event Transport** - Weddings, parties, special occasions
4. **Urban Rides** - City transportation on-demand
5. **Tourist Tours** - Marseille and PACA region discovery

### Vehicle Fleet

- Mercedes E-Class (Premium sedan)
- Mercedes S-Class (Luxury sedan)
- Mercedes Vito (Premium minivan - 8 passengers)
- BMW 5 Series (Sport sedan)
- Audi A6 (Executive sedan)
- Tesla Model S (Electric luxury)

### Professional Drivers

- Experienced drivers
- Multilingual (French, English, Arabic, Italian, Spanish)
- Professional license and safety training
- Local area expertise
- Discreet and punctual service

## Technologies Used

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **i18next** - Internationalization (French/English)
- **Heroicons** - Icons
- **React hooks** - State management

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
src/
  app/
    - page.tsx (Homepage)
    - services/ (Service pages)
    - vehicles/ (Vehicle showcase)
    - contact/ (Booking/Contact)
  Components/
    - LandingPage/ (Homepage sections)
    - Navigation/ (NavBar, Footer)
    - Button/ (Reusable buttons)
  Context/
    - Language/ (i18n context)
    - Animation/ (Scroll animations)
public/
  - locales/ (French/English translations)
  - img/ (Images for vehicles, services, drivers)
```

This transformation converts the original textile/packaging business website into a professional VTC service website optimized for the Marseille market.
